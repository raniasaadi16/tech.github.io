JIRA TRAINING FOR BUSINESS ANALYSTS

Course Overview

Jira training course is designed and developed for Business Analysts and project managers. This is a foundation level course. It touches the basics of managing requirements in Agile projects, basics of Agile project management and using other features of Jira. This is a self-paced Jira certification course and includes quizzes and assignment. You will be given a certification based on a course completion test.

  • Key Features of Jira
  • Using various features of Jira
  • Workflow in Jira
  • Agile Project management in Jira
  • Using Jira for Scrum
  • Using Jira for Kanban
  • Using Reports in Jira
  • Using Filters in Jira

  • It is an ideal course for business analysts at any stage of their career to understand the Jira tool
  • It can also be taken up by any Project Management or technical professional who needs to use Jira in the project

  • Get an understanding of the JIRA tool and environment
  • Get hands-on and practical knowledge of using Jira for requirements handling in Agile
  • Get enabled in creating ad-hoc reports for customers on-demand

Course Syllabus

  • Jira Introduction
  • How to use Jira
  • Workflow in Jira

  • Implementing Scrum in Jira
  • Implementing Kanban in Jira
  • Jira Reports
  • Working with filters in Jira
